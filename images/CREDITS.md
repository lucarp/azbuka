# Illustration credits

The card illustrations in this folder are from **OpenMoji** — the open-source
emoji and icon project.

- Source: https://openmoji.org
- License: **CC BY-SA 4.0** (https://creativecommons.org/licenses/by-sa/4.0/)
- Color SVG set, fetched via jsDelivr (`openmoji@15.0.0`).

Files are unmodified OpenMoji color SVGs, renamed to plain English slugs
(e.g. `1F415.svg` → `dog.svg`) for use as flashcard images in this app.
